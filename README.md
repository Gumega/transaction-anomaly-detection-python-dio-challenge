# Transaction Anomaly Detection in Python | DIO Challenge

This project was made for the Transaction Anomaly Detection in Python challenge from DIO's Accenture - Python para Análise e Automação de Dados bootcamp.
The objective is to use the best possible approach to identify/detect with precision transaction's anomalies from the CSV file: https://storage.googleapis.com/download.tensorflow.org/data/creditcard.csv.

With this code, I was able to reach the following result:
